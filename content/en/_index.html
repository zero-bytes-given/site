What is Probr?

An open-source automated testing framework for an ecosystem of community-driven test packs to prove compliance validation in the cloud, on-prem, pre-prod and prod environments alike. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-wrench" title="Validate any Resource" url="./docs/guides/sdk/" %}} Using the Probr SDK, service packs can be built to target any resource, test against any standard, and integrate with other service packs for a seamless runtime! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-chart-area" title="Logs and Summaries" url="./docs/guides/output/" %}} Every service pack can produce harmonized runtime logs and summaries in JSON format, in addition to Cucumber output that is ready to be transformed into beautiful visuals! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-laptop-code" title="Build and Harness" url="./docs/getting-started/" %}} Build your own compliance validation pack, or harness a service pack that the community has already built. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="secondary" >}} {{% blocks/feature icon="fab fa-slack" title="Join us on Slack" url="https://join.slack.com/t/probrworkspace/shared_invite/zt-1bcag5m3b-P5XMEoZ1imlMAcY0KpNkZQ" url_text="Chat with us" %}} Join the community on Slack {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/probr" url_text="Contribute to Probr" %}} We follow the [Pull Request](https://docs.github.com/en/github/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests) contributions workflow on **GitHub**.
